Markdown 使用笔记

2022.8.31

Markdown 段落格式

标题

n 个 # + (空格) = n级标题

n = 1~6 , Markdown中最多可以表示 6 级标题

字体

_ 斜体字 _ (或者是*斜体字*)： 斜体字

** 粗体字 ** (*同样可以用_来替代，数量一致就行)： 粗体字

但是个人还是喜欢将 * 和 _ 分开，因为会遇到接下来的这位仁兄：

_** 又斜又粗字体 **_ （当然也是可以互换符号位置，或者是连用三个相同符号）： 又斜又粗字体

**我收回上面的话，还是只用 * 吧，不然切换输入法好烦。。。 **

分割线

*** 可以是分割线

--- 也可以是分割线

如果正在写的东西让你足够暴躁，当然可以多打大于 3 个的 * 和 - ,否则还是省点力气吧，节约一下胃里还没被消化的食物。

删除线 and 下划线

~~ 删除线 ~~: 删除线

<u> 下划线 </u>: 下划线

加入 假如你是个想要吸引别人注意的 错别字大王，那么这招有用。

注脚

注脚： [^ 要注明的文本]

注脚的内容： [^要注明的文本]： 巴拉巴拉巴拉

例如： 注脚在右上角^鼠标移到这里查看内容

注脚的内容一般放在文章的最后，不然让别人第二行就可以看到的东西为什么要多此一举打个注脚？此外我个人认为注脚的用法其实可以这样。^惊喜

Markdown 列表

无序列表

  • * + (空格) 可以
  • + + (空格)也可以
  • - + (空格)还可以

有序列表

  1. 数字 + . + (空格)

  2. 回车后自动会续上下一个序号的列表

  3. 在 Typora 中如果数字不连续会间隔稍微大一点

  4. 比如 2 和 4 之间

列表嵌套

  1. 列表嵌套可以在母列表完成后换行

    然后回车掉前面的数字（在Typora中可以这样）

  2. 也可以在子列表的列表种类（有序和无序都可以）前添加两个或四个空格

    • 就可以嵌套了

Markdown 区块

引用区块

> + （空格） + 内容 而且此行之前没有其他内容

就是引用块了，斜体、粗体、_**又粗又 黑 斜而且有 删除线 和下划线的内容都可以用在其中。 **_

如果要引用一大段诗句，则每行都要加 > 符号，在每行结束之后增加 两个空格 在换行可以使得每行缩进相同。

Markdown 链接

一般链接

[链接名称](链接地址) 或者是 <链接地址>

如：百度的链接是 百度一下

​ https://www.baidu.com

需要注意的是括号是中文还是英文，只有英文括号才行。

高级链接

如果链接要出现多次，则可以用一个变量来赋值一个链接，再将链接放在文章最后

如：

链接(Baidu)用 1 作为网址变量： [Baidu][1]
链接(Bing)用 2 作为网址变量： [Bing][2]

在文章末尾进行赋值

[1]: https://www.baidu.com

[2]: https://www.bing.com

Baidu Bing

图片

图片也是一种链接！是不是很惊喜！所以要在 [ ] 之前加一个 ！

![图片名称](图片链接”图片名称”)

！看得我也好想睡觉！可恶！

当然，图片也可以使用高级链接

Markdown 表格

可以使用 | 来分割不同的单元格，表头可以用 - 来分隔

| 表头 | 表头 |

| —- | —- |

| 单元格 | 单元格 |

|单元格 | 单元格 |

表头 表头
单元格 单元格
单元格 单元格

内容和标题栏的对齐方式，也通过对分隔行添加 ： 可以也可以设置

右对齐： -:

左对齐： :-

居中： :-:

其他

代码

在 markdown 中写代码也未尝不是一种方式，虽然没有在编译器里写那么舒服

代码（当然也可以是函数）的片段可以用反引号 ‘ ` ‘ 来包起来

譬如： `print()` 在 python 中是打印的意思

​ print() 在 python 中是打印的意思

与编译器中相同，Tab 键也表示四个空格，代码区块可以用 Tab 来分隔

此外，可以用 ' ``` ' 来包裹一段代码，指定语言（也可以不指定）

```Python

for i in range(10):

​ print(‘Hello World’)

转义

在 Markdown 中，在 ‘ \ ‘ 之后的字符可以正常显示，不会体现特殊意义，’ * ‘ 等字符都可以使用

公式

Markdown 中使用 Tex 来渲染数学表达式，Typora 中也可以选择使用 LaTax ，分隔符有两种可以选择：

  1. $ … $ 与 *\( … )\* 中的公式将在行内显示

  2. $$ … $$ 与 *\[ … ]\* 以及 ```math 中的公式将在块内显示

  1. $ f(x) = sin(x) $

  2. f(x) = sin(x)
